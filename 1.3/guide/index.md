# switchable

## 注意

本模块仅用于兼容 <=1.3 的应用代码。新应用推荐使用 ``gallery/slide`` .


## API

参考 

http://docs.kissyui.com/docs/html/api/component/switchable/switchable.html

引用方式换做：

``` javascript

	KISSY.use('gallery/switchable/1.3/',function(S,Switchable){
	    // use Switchable
	});
